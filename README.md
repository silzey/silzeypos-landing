# SilzeyPOS Landing Page

This is a Next.js project for the SilzeyPOS landing page, built with Firebase Studio.

## Getting Started

To get started with development, run the development server:

```bash
npm run dev
```

This will start the Next.js application in development mode.

## Deployment

This project is configured for continuous deployment to Firebase Hosting. Any push to the `main` branch on GitHub will automatically trigger a build and deploy the latest version of the application.
